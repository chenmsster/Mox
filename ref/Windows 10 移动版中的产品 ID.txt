获取其他应用的产品 ID 和 AUMID

若要获取从 Windows 应用商店安装的或本地安装（旁加载）的应用的产品 ID 和 AUMID，请使用以下步骤。
先决条件：设备中插入了 SD 卡，并且安装了所有要获取 ID 的应用
在“开始”屏幕  上，轻扫到“应用列表”，然后依次点击“设置” >“帐户”****>“应用园地”。
点击“应用”****、点击以选择你想要获取 ID 的应用，然后点击“完成”
点击“高级”，然后点击“导出到 SD 卡”****。
使用 USB 将设备连接到电脑，然后在设备的 SD 卡上打开 WEHLockdown.xml 文件，查看每个应用的产品 ID 和 AUMID。
